🔍 Key Features:

• Highlights attribute keywords (e.g., value, text, visible) with customizable foreground color and italic style

• Applies a distinct background color to the entire value for visual separation

• Styles comments inside the binding expression (e.g., /* comment*/ or // comment ) with a separate color

• Highlights brackets ({}, [], ()) to improve readability of nested bindings

• Optionally renders the attribute name in bold

• Fully customizable via Tools > Options > Knockout Highlighter

🎯 Designed for developers using Knockout.js in:

• .html, .htm, .cshtml, .vbhtml, and other Razor-based views

• ASP.NET MVC or legacy Knockout-based projects

🛠 Built with performance and clarity in mind, this extension integrates seamlessly with Visual Studio’s classification system.
